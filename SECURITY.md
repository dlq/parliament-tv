# Security

This is an early public-interest prototype. It does not currently include accounts, payments, analytics, a server-side sync layer, or push notifications.

Please do not open a public issue for sensitive reports involving:

- credential, token, or secret exposure;
- a vulnerability in future telemetry, sync, or CloudKit-backed features;
- unsafe handling of external pages, streams, or player embeds;
- a privacy issue that could identify app users or viewing behaviour.

For now, report sensitive issues privately to the repository owner through GitHub. Public playback bugs, broken stream links, and source corrections can use normal GitHub issues once issues are enabled.

Do not test against parliamentary, broadcaster, video-platform, or streaming-vendor systems beyond ordinary public access checks.
